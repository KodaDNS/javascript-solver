# javascript-solver
- Simple script to reduce obfuscation & decode complex patterns

# Features
- Supports unlimited files in the input
- Supports multiple processes at the time
- Supports large files

Please note that with obfuscation, only wrappers (i.e., strings) can be reduced. Origin names are random and cannot be reversed. This tool only offers the ability to unravel specific patterns, obfuscate, decode encoded strings, and more.
